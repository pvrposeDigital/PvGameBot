Time to add PVRPOSE to your server.

This bot is designed to play the Pvrpose Card Game with friends on Discord. It allows users to join the game, draw prompt cards, select a question, and have another player answer it for a fun experience.

Add To Discord Premium (COMING SOON)

Features

Feature # JOIN

Join the game by typing !join

Feature # DRAW

Draw a prompt card with two random questions by typing !draw

Feature # TOP or BOTTOM

Choose one of the questions by clicking Top or Bottom

Feature # MENTION USER

The bot will mention another user in the chat to answer the selected question
